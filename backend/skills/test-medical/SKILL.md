# 医疗规划测试技能

这是一个测试技能，用于验证技能系统是否正常工作。

## 触发条件

当用户提到以下关键词时自动激活：
- 患者、病历、治疗、处方、药品、诊断、检查、科室、搜索、填写、点击

## 功能

1. 辅助填写患者信息表单
2. 辅助搜索医疗记录
3. 辅助制定治疗方案

## 使用说明

此技能为测试技能，主要用于验证：
1. 技能加载是否正常
2. 技能匹配是否准确
3. 技能内容是否注入到 Agent 提示词中
4. UI 是否正确显示技能激活状态
